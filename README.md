# Daily Report

このリポジトリは、日々の作業内容・学習ログ・気づきを記録するための日報リポジトリです。

毎日少しでもアウトプットを積み重ね、成長の過程を可視化することを目的としています。

## 目的

- 毎日コミットする習慣を作る
- 学んだことを言語化して定着させる
- 日々の作業内容を振り返る
- 技術的な成長の軌跡を残す

## 日報の書き方

`MM-DD.md` の形式でファイルを作成し、日々の記録を書き残します。
