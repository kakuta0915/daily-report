## 学んだこと・気づいたこと

#### リーダブルコード

- 式が巨大なときは分割する。
  - 説明変数や要約変数を利用し、式の説明をする。
- グローバル変数は名前空間の汚染の原因になるため使用しない方が良い。

#### React 復習

- ノードには、要素ノード・属性ノード・テキストノードの 3 種類がある。
  ```jsx
  element
  ├─ type: "h1"         ← 要素ノードに相当
  └─ props
  ├─ title: "foo"       ← 属性ノードに相当（ただしJSでは props として保持）
  └─ children: [
  		{
                    type: "TEXT_ELEMENT", ← テキストノード
                    props: {
                    nodeValue: "こんにちは",
                    children: []
                  }
  	      }
  	    ]
  ```
- テキストノードの処理と属性ノードの処理は別々で記述しなければならない。
- React 18 以前の同期レンダリング
  - 1 つの更新が始まると、更新が完了するまで止まらない。
  - 描画処理は途中で中断できない。
- React 18 以降の並行レンダリング
  - レンダリングを中断できる
  - 優先度の高い処理に割り込める
  - バックグラウンドでレンダリングできる
