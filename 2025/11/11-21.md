## 学んだこと・気づいたこと

- 仮想 DOM とは、React や Vue.js に備わっている仕組みで、コンポーネントの state や props が変化した際に、まずメモリ上で新しい仮想 DOM を構築する。
  そして、前回の仮想 DOM と比較（diff）して差分を抽出し、必要な部分だけを実際の DOM に反映（patch）することで、効率的に画面を更新する。
- HTML ライクとは、React の jsx 内で、見た目が HTML に似ていることを意味する。
