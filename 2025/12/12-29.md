## 学んだこと・気づいたこと

- TypeScript における `Promise<T>` の意味を正確に理解した。
  - `params: Promise<{ slug: string }>` の場合、`params` 自体はオブジェクトではなく Promise である。
  - `await params` して初めて `{ slug: string }` のようなオブジェクトを扱える。
  - 「params はオブジェクト」と説明すると誤解を招くことを認識した。
- `fetch` を使った非同期処理の流れを整理できた。
  - `fetch()` は `Promise<Response>` を返す。
  - `res.json()` も Promise を返すため、非同期処理が 2 段階ある。
  - `await` を付けることで、最終的に JSON をパースしたデータを取得できる。
- 関数の引数に書く `type?: string` は 省略可能な引数 を表すことを理解した。
- `Array.includes(value)` は、配列の中に値が含まれているかを `true / false` で判定するメソッドだと理解した。
