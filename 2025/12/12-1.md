  これらは React において何もレンダーされない値 として扱われる。

## メモ

#### **Q1：条件が null のとき、レンダーされる？**

→ **多くの場合レンダーされません（画面に何も出ない）**

ただし、**三項演算子なら「表示する側」を返せば表示されます。**

#### **Q2：条件が false のとき、レンダーされる？**

→ **false はレンダーされません。**

React は false / null / undefined を **何も描画しない値** として扱います。
