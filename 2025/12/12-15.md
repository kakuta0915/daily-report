## 学んだこと・気づいたこと

- XAMPP のローカルホストが `http://localhost` なんだけどセキュリティ的に大丈夫？

  - `http://localhost` は外部からアクセス出来ないのでセキュリティ的に問題ない。
  - `localhost` ＝ **自分の PC 自身**でネットワークに出ていない。インターネットを通らない。

- メタデータ（metadata）とは、「データそのものを説明するためのデータ」です。

#### なぜメタデータが重要なのか

メタデータは、

- **検索エンジンにページ内容を正しく伝える**
- **SNS での表示をコントロールする**
- **ユーザーのクリック率に影響する**

という点で、**中身と同じくらい重要**です。

本文がどれだけ良くても、メタデータが適切でなければ

- 検索結果で目立たない
- SNS で変なタイトルや画像が出る

といった問題が起こります。

#### layout.tsx にメタデータを書く理由

`layout.tsx` は **全ページの親**です。

そのため、

- サイト共通のタイトル
- 共通の description
- 共通の OGP 設定
- favicon

といった **「どのページにも共通するメタデータ」** を置くのに最適です。

個別ページでは、必要に応じて上書きします。

```tsx
// app/articles/[slug]/page.tsx
export const metadata = {
  title: "記事タイトル",
};
```

この場合、

- layout の設定をベースに
- page の metadata が部分的に上書きされる

という挙動になります。
